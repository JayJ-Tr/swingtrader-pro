🤖 AI Proxy Setup
Default (no setup needed): Uses corsproxy.io automatically — free, works immediately.

Custom proxy (optional, more reliable):
1. Go to dash.cloudflare.com → Workers & Pages → Create Application → Create Worker
2. Replace default code with the code below → Deploy
3. Workers Settings → Variables & Secrets → add: ANTHROPIC_KEY = your Anthropic key
4. Copy the Worker URL → paste in Proxy URL field above

export default { async fetch(req, env) { if (req.method === 'OPTIONS') return new Response(null, { headers: { 'Access-Control-Allow-Origin': '*', 'Access-Control-Allow-Headers': 'Content-Type', 'Access-Control-Allow-Methods': 'POST' } }); const body = await req.json(); const res = await fetch('https://api.anthropic.com/v1/messages', { method: 'POST', headers: { 'Content-Type': 'application/json', 'x-api-key': env.ANTHROPIC_KEY, 'anthropic-version': '2023-06-01', 'anthropic-beta': 'messages-2023-12-15' }, body: JSON.stringify(body) }); const data = await res.json(); return Response.json(data, { headers: { 'Access-Control-Allow-Origin': '*' } }); } };
4. In your Worker settings → Variables → add secret: ANTHROPIC_KEY = your Anthropic API key
